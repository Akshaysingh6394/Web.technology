HTML <a> Tag

The <a> (anchor) tag defines a hyperlink, which is used to link from one page to another. The most important attributes of the <a> tag are:

  • href: Specifies the URL of the page the link goes to.
  • target: Specifies where to open the linked document (e.g., _blank for a new tab).
  • title: Specifies extra information about the link (displayed as a tooltip when hovering).

Example of an <a> Tag

In this example:

  • href is set to https://www.google.com, so clicking the link will take the user to that website.
  • target is set to _blank, so the link will open in a new tab.
  • title is set to Go to Example, so hovering over the link will show this tooltip.
